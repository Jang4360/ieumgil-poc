# ieumgil-osm-etl-poc

## 목적
OSM 원천 데이터 확보, 범위 검증, 향후 보행 네트워크 전처리를 위한 PoC 프로젝트다.

## 현재 단계
현재는 부산 OSM 원천 데이터 확보 및 범위 검증만 수행한다.

## 현재 단계에서 하지 않는 것
- 본격적인 ETL 구현
- DEM 기반 경사도 계산
- 공공데이터 병합
## 원천 데이터

- 파일명: busan.osm.pbf
- 출처: BBBike Extract
- 범위: 부산광역시 영역
- 포맷: OSM PBF

## 비고
- 샘플 검증용 데이터로 사용
- 향후 Geofabrik 기반 전체 데이터 검토 예정
