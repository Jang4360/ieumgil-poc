# 04_WALK_SEGMENT_EXTRACTION_VALIDATION.md

## 단계명

대중교통 경로의 도보 구간 추출 가능성 검증

## 목적

대중교통 후보 경로에서 출발 도보, 환승 도보, 도착 도보 구간의 시작점 / 종료점을 식별할 수 있는지 확인한다.

## 구현 접근

1. Step 2에서 확보한 경로 응답을 구간 단위로 분해한다.
2. 도보 타입 구간이 분리되는지 먼저 확인한다.
3. 각 도보 구간의 시작 anchor와 종료 anchor가 정류장 / 역사 / 출입구와 연결되는지 본다.
4. 출발 도보, 환승 도보, 도착 도보를 구분하는 규칙을 정리한다.
5. 역사 내부 이동처럼 외부 좌표만으로 해석하기 어려운 케이스를 별도 분리한다.

## 확인할 것

- 도보 구간이 응답에서 별도 타입으로 식별되는지
- 도보 구간 시작 / 종료 지점 좌표를 얻을 수 있는지
- 환승 도보와 일반 도보를 구분할 수 있는지
- 정류장 / 역사 / 출입구 좌표와 연결 가능한지
- 구간 수가 늘어나도 동일 규칙으로 해석 가능한지

## 산출물

- 도보 구간 분해 규칙 문서
- 샘플 경로별 도보 anchor 추출 결과
- 출발 / 환승 / 도착 도보 구분 기준표
- 해석 불가 또는 애매한 케이스 목록

## 완료 기준

- 최소 1개 이상 경로에서 도보 구간 시작점 / 종료점 식별
- 출발 도보 / 환승 도보 / 도착 도보 구분 기준 정리
- 내부 이동 등 후속 검토 대상 케이스 분리

## 범위 제한

- 역사 내부 세부 동선 복원은 현재 단계에서 하지 않는다.
- 실내 경로 탐색이나 지하 연결통로 모델링은 포함하지 않는다.
