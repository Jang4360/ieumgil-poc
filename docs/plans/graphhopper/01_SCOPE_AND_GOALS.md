# 01_SCOPE_AND_GOALS.md

## 목적

부산 지역 OSM 원천 데이터를 확보하고, 보행 네트워크 구축에 사용할 수 있는지 확인한다.

---

## 확인할 것

- 부산 범위 데이터 확보 가능 여부
- 파일 포맷 확인 (`.osm.pbf`)
- 보행 관련 태그 포함 여부
- 데이터 크기와 처리 가능성

---

## 완료 기준

- 부산 범위 OSM 파일 1개 이상 확보
- 보행 태그 샘플 확인
- 적재 대상 파일 확정

---

## 범위 제한

현재 단계는 원천 데이터 확보와 검증에만 집중한다.

다음은 현재 범위에서 제외한다.
- ETL 구현
- GraphHopper import
- DB 적재
- 경사도 계산
- 공공데이터 병합

---

## 기대 결과

이번 단계가 끝나면 다음 단계에서 아래 작업을 시작할 수 있어야 한다.

- OSM 원천 파일을 기준으로 샘플 구역 추출
- 보행 가능 태그 필터링 기준 수립
- GraphHopper import 대상 파일 지정
