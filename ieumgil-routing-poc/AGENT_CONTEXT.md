# AGENT_CONTEXT.md

## 프로젝트 역할
이 프로젝트는 Spring Boot 기반 PoC 프로젝트다.

## 현재 단계 역할
현재 단계에서는 아래만 준비한다.

- Spring Boot 프로젝트 초기 구조 유지
- 이후 GraphHopper 연동을 위한 최소 환경 준비
- Postgres 연결 예정 구조 고려

## 현재 단계에서 하지 않는 것
- 비즈니스 로직 구현
- 엔티티 본격 작성
- API 명세 확장
- Slack 반영 로직 구현
